# ASR 错词模式与校正参考

> Whisper ASR 常见错词规律，整理稿校正时优先处理。

---

## 1. 人名校正

| 原文 | 常见错写 |
|------|---------|
| 张小龙 | 张孝龙、张晓龙 |
| 卡兹克 | 卡兹客、卡兹克 |
| OpenAI | Open AI、openai |
| 巴菲特 | 巴菲、芭菲 |

**处理方式：** 人名优先对照原文或公开资料确认，不确定时保留 ASR 结果并加注。

---

## 2. 公司名 / 产品名校正

| 场景 | 常见错写 |
|------|---------|
| 字节跳动 | 字节title、字节条 |
| 微信 | 微微信、微信息 |
| ChatGPT | Chat GPT、chatgpt |
| LLM | L-L-M |

---

## 3. 技术术语

| 正确 | 常见错写 |
|------|---------|
| API | A-P-I、A-P-I接口 |
| token | Token、令牌 |
| Prompt | prompt、提示词 |
| RAG | R-A-G |
| fine-tuning | finetuning、 微调 |

---

## 4. ASR 噪音清洗

- 无意义字符：`&*%#@` → 删除
- 乱码片段：长度 < 5字且无语义 → 删除
- 重复句：连续重复超过2次 → 保留一次
- 语气词堆叠：`嗯嗯嗯、啊对对对` → 酌情删减

---

## 5. 数字 / 年份校正

- 数字容易连读错读，如"2024年"听成"二霌二四年"
- 金额、品牌型号等数字优先保留原文或上下文推断
- 不确定时保留 ASR 结果并加 `[?]` 标注

---

## 6. 置信度低时的处理原则

- 置信度 < 0.7 的片段 → 保留原词 + 加注，不直接删改
- 人名/公司名/产品名不确定 → 搜索核实后校正
- 专业术语不确定 → 不改，加 `[?]` 标注
